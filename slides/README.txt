Medical Appointment System - Glass Style HTML Presentation

How to use:
1. Open the folder in VS Code.
2. Open index.html with Live Server or double-click it in your browser.
3. Use Next / Previous buttons or keyboard arrows to move between slides.

Where to add screenshots:
- Slide 3: System Overview Screenshot
- Slide 6: Data Structure Diagram (optional; current diagram is already made with HTML/CSS)
- Slide 8: Menu Screenshot
- Slide 9: Doctor List, Insert, Update, Delete, Appointment List, CSV File screenshots

In the HTML file, you will find comments like:
<!-- INSERT SCREENSHOT HERE: ... -->
Replace the placeholder box with an image tag, for example:
<img src="assets/menu-screenshot.png" alt="Menu Screenshot">
